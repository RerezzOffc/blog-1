How to Deploy Node.js with Heroku

Published 3 months ago5 min read4 comments

I recently created a small app using Node.js. Then, we deployed the app on Heroku, and I was so impressed by the blazing-fast experience. Here are the few simple steps to follow to deploy your Node.js application:

Prerequisite: Heroku CLI, Git Bash.

1. Login to Heroku

In your terminal run the following command. Make sure that you have in the right directory that you want to deploy. By running this command you can login to your Heroku account using your credentials.

2. Initialize the repository with git

If the repository is not initialized with git. Initialized it with git. and make sure that you have committed everything in that repository.

					    
$ git init

This will initialize the repository with git.

					    
$ git add .

This command will add every file and folder that is there in the following repository to the staging area.

					    
$ git commit -m “commit message”

This command will commit the files in the staging area.

3. Create the app on Heroku

In your terminal, place yourself inside your app folder. Make sure everything is pushed to your GitHub or other remote repo before you begin.

Run the command:

By running this command, a git remote is created associated with your local repository. This also generates a random name and URL(where the app is going to be deployed) for your app.

4. Tell Heroku to run “node app.js”

Then we need to add a script in our package.json file to tell Heroku to run node app.js. Just add the line "start": "node app.js” to your "script" object in your package.json: Make sure that the file is saved. Then you need to commit the file again (Just type 3rd command in step 3).

5. Push the master

If “git push heroku main” does not work try doing it with “git push heroku master”. It might not work with some devices. This command will push everything to the Heroku master repo.

					    
$ heroku logs

By using this command, you will get the app’s most recent logs.

***************************************************************************************

Hurray!! Now your app is deployed and you can check the app in the link that Heroku has provided. While pushing the application into Heroku you may get some error. Check the following steps carefully and try it again.
